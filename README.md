### 介绍

喵喵是一个设计稿管理的平台，V0.1版本核心在于做RP导入稿的管理（解决因axshare服务慢的问题）。

### 核心概念与功能说明

* 注册/登录/忘记密码
* 角色定义与权限系统
* 资源聚合与控制模型
* 系统初始化与操作流程
* 设计稿上传/查看/关注/下载/标注/分享/版本历史/查看记录/点评/删除
